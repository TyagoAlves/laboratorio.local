Componentes

Tudo que você precisa
para montar seu laboratório

Do Active Directory ao acesso remoto via navegador — cada serviço foi pensado para ser modular, seguro e fácil de gerenciar.

Active Directory

Domínio laboratorio.local com AD DS, DNS integrado, OUs organizacionais e grupos de segurança. Tudo configurado automaticamente via PowerShell.

Apache Guacamole

Acesso remoto via navegador sem instalar clientes. Autenticação LDAP contra o AD com suporte a grupos aninhados e permissões granulares.

E-mail Corporativo

hMailServer com SMTP, IMAP e suporte a TLS. Instalado automaticamente dentro do Windows Server. Pronto para configurar domínios e contas.

Windows Server 2022

Roda via KVM dentro do Docker (dockurr/windows). Setup 100% automatizado: AD, DNS, WinRM e compartilhamento SMB. Acessível via RDP e web UI.

Linux Desktop

Desktop Ubuntu com ambiente LXDE acessível via VNC ou diretamente pelo Guacamole. Perfeito para testar conectividade e políticas do AD.

Cloudflare Tunnel

Exponha o Guacamole publicamente sem abrir portas no firewall. Tunnel gerenciado pelo Cloudflare com perfil opcional no Docker Compose.

Quick Start

Rodando em 10 minutos

Pré-compilado para Linux. Só executar.

Instalação via Linux

✓ Pré-requisitos:
sudo apt update && sudo apt install -y docker.io docker-compose-v2 git
✓ Verifique o KVM:
sudo kvm-ok

✓ Clone e execute:
git clone https://github.com/TyagoAlves/laboratorio.local.git
cd laboratorio.local
docker compose up -d

✓ Acompanhe o setup:
docker compose logs -f windows

Requer: KVM habilitado no kernel. Ubuntu: sudo kvm-ok

Problemas com a instalação? Abra uma issue no GitHub
